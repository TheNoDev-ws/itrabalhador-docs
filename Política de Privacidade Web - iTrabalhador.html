Política de Privacidade

Aplicativo: iTrabalhador

Última atualização: 28 de Novembro de 2025

1. Introdução

Esta Política de Privacidade descreve como o iTrabalhador trata dados inseridos pelo usuário. O aplicativo foi desenvolvido para funcionar localmente, priorizando a privacidade e a segurança, sem criar contas obrigatórias ou enviar informações para servidores próprios sem o consentimento do usuário.

2. Dados Coletados

O aplicativo coleta e armazena os seguintes dados, de forma direta ou indireta, para viabilizar suas funcionalidades:

  • Dados de Simulações: Valores de salário (bruto/líquido), benefícios, dependentes, custos, datas de admissão/afastamento e informações relacionadas a cálculos trabalhistas (férias, 13º, rescisão, horas extras). Estes dados são gravados localmente via SharedPreferences apenas se o usuário optar por salvá-los.
  • Preferências e Configurações: Escolhas de tema (claro/escuro), modo avançado e tabelas personalizadas de INSS e IRRF.
  • Registro de Ponto: Horários de entrada/saída, tipos de marcação, notas, configurações de jornada, justificativas e banco de horas. Estes dados são armazenados em banco de dados local (Hive).
  • Feriados Locais: Datas comemorativas criadas manualmente pelo usuário.
  • Documentos PDF: Relatórios gerados a partir dos dados acima. O processamento é local.

Nota sobre Dados Sensíveis: O app não solicita identificadores oficiais (CPF, RG) ou endereço. Informações financeiras e de rotina de trabalho permanecem restritas ao armazenamento local do dispositivo.

3. Finalidade do Uso dos Dados

Os dados são utilizados estritamente para:

  • Realizar cálculos trabalhistas e simulações financeiras.
  • Gerenciar o controle de ponto eletrônico pessoal.
  • Personalizar a experiência de uso (temas e alíquotas).
  • Gerar relatórios em PDF para uso pessoal.

4. Compartilhamento e Terceiros

O iTrabalhador não realiza envio automático de dados para servidores externos. Não há integração com serviços de análise, publicidade (AdMob) ou bancos de dados em nuvem (Firebase Cloud Firestore) para sincronização automática.

O compartilhamento de dados ocorre apenas se o usuário decidir exportar um relatório (PDF) e enviá-lo manualmente através de outros aplicativos instalados no dispositivo (e-mail, WhatsApp, Telegram, etc.).

5. Armazenamento e Segurança

Todos os dados residem no armazenamento interno do dispositivo do usuário. O aplicativo utiliza tecnologias padrão de persistência local (SharedPreferences e Hive).

Atenção: Estes métodos de armazenamento padrão não aplicam criptografia forte por padrão. Recomenda-se que o usuário mantenha seu dispositivo protegido por senha, PIN ou biometria para garantir a segurança das informações em caso de perda ou furto do aparelho.

6. Controle e Exclusão de Dados

O usuário possui total controle sobre seus dados:

  • Edição e Exclusão: É possível editar ou remover simulações, registros de ponto e feriados através da interface do aplicativo.
  • Reset: O usuário pode restaurar as tabelas tributárias aos valores padrão.
  • Exclusão Total: A desinstalação do aplicativo remove permanentemente todos os dados armazenados localmente pelo iTrabalhador.

7. Permissões do Dispositivo

O aplicativo não solicita permissões sensíveis (como Câmera, Localização ou Contatos). A funcionalidade de gerar e compartilhar PDF utiliza os mecanismos nativos de "Intenção de Compartilhamento" do sistema operacional Android, não exigindo permissões especiais de armazenamento no manifesto para as versões mais recentes do Android.

8. Crianças e Menores de Idade

O aplicativo não é direcionado a crianças e não coleta intencionalmente dados de menores. Recomendamos que usuários menores de idade utilizem a ferramenta sob supervisão de um responsável.

9. Contato

Em caso de dúvidas sobre esta política ou sobre o tratamento de dados no aplicativo, entre em contato através dos canais de suporte informados na página do aplicativo na loja.
